# monitor-test

A bootable monitor test program for PCs.

## Instructions

- Assemble and link.
- Write *test* to a disk.
- Boot from the disk.
- Press any key to step through the following colours:
  * White
  * Yellow
  * Magenta
  * Light Red
  * Light Cyan
  * Light Green
  * Light Blue
  * Dark Grey
  * Light Grey
  * Brown
  * Purple
  * Red
  * Cyan
  * Green
  * Blue
  * Black
